# 数据采集协议 v1.0

## 一、硬件配置

| 参数 | 设定值 | 依据 |
|------|--------|------|
| 主控 | ESP32-WROOM-32 | 支持MicroPython，内置WiFi/BLE |
| IMU传感器 | MPU6050 | 6轴（加速度+陀螺仪） |
| 地磁传感器（可选） | HMC5883L | 提供航向角 |
| 采样率 | **50 Hz** | 人体运动频率<20Hz，满足奈奎斯特定理 |
| I2C频率 | 400 kHz | 标准高速模式 |

## 二、传感器佩戴

| 佩戴部位 | 固定方式 | 坐标轴定义 |
|---------|---------|-----------|
| **腰部**（推荐） | 腰带包/弹性腰包 | X:前进方向，Y:右侧，Z:垂直向上 |

> 要求：全流程佩戴位置固定，拍照记录并存档

## 三、被试方案

| 项目 | 要求 |
|------|------|
| 被试人数 | ≥ 3 人 |
| 被试编号 | S01, S02, S03（固定且记录） |
| 每类每人次数 | ≥ 3 次 |
| 每次采集时长 | ≥ 60 秒（稳定活动段） |
| 被试基本信息 | 记录身高、体重、性别（可选） |

## 四、活动类别集

| 编号 | 类别名 | 说明 | 动作描述 |
|------|--------|------|---------|
| 0 | 跌倒 | 从站立到倒地的完整过程 | 站立→前倾/侧倾→倒地（软垫保护） |
| 1 | 走路 | 正常速度平地行走 | 自然步态，速度约1.2-1.5 m/s |
| 2 | 跑步 | 中速跑步 | 速度约2.5-3.5 m/s |
| 3 | 静坐 | 静止坐在椅子上 | 身体放松，无明显动作 |

## 五、打标方式

### 5.1 时间戳打标
- 开始采集：按下物理按钮（或手机App）+ 语音口令记录
- 结束采集：同样方式记录时间戳
- 在实验记录本上记录每次的开始/结束时间

### 5.2 数据文件记录
每个CSV文件第一行记录元数据：
